## Restaurant Performance Dashboard (Excel Project)

## Project Overview
This project focuses on analyzing restaurant data using Microsoft Excel.  
The goal of the project is to create an interactive dashboard to analyze restaurant performance based on rating, cuisine, location, offers, and pricing.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## Dataset Columns
- Restaurant Name
- Cuisine
- Rating
- Number of Ratings
- Average Price
- Number of Offers
- Offer Name
- Area
- Pure Veg
- Location
- Date

## Dashboard Features
- Total Restaurants KPI
- Average Rating KPI
- Cuisine Wise Restaurant Analysis
- Area Wise Restaurant Count
- Pure Veg vs Non Veg Distribution
- Offer Analysis

## Insights Generated
- Most popular cuisine types
- Areas with the highest number of restaurants
- Restaurants offering the most discounts
- Average ratings across cuisines
